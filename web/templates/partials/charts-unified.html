Rentabilité pure : Gain ou perte nette si vous revendez le bien Ne comptabilise pas l'épargne forcée (capital remboursé)
Comment lire ce graphique ?

Objectif

Visualiser votre gain ou perte nette si vous revendez le bien après X années.

Formule de calcul

Plus-value nette = Appréciation du bien - Coûts irrécupérables

Coûts irrécupérables = Intérêts + Assurance + Frais (notaire, agence, dossier) + Taxe foncière + Charges copro + Entretien

Comment lire

  • Zone verte : Vous êtes rentable (appréciation > coûts)
  • Zone rouge : Vous perdez de l'argent (appréciation < coûts)
  • 3 scénarios : -1%/an (baisse), 0%/an (stable), +1%/an (hausse modérée)
Attention : Pour être rentable, l'appréciation du bien doit dépasser les coûts irrécupérables. Avec un prêt classique, il faut souvent +2%/an ou plus pour atteindre la rentabilité.

Patrimoine total : Richesse nette accumulée Inclut l'épargne forcée via remboursement du capital
Comment lire ce graphique ?

Objectif

Comparer le patrimoine accumulé entre rester locataire ou acheter, année après année. La courbe la plus haute gagne.

Formules de calcul

Locataire = Apport placé × (1 + taux)^n + Épargne mensuelle placée - Loyers cumulés
Acheteur = Valeur du bien - Capital restant dû - Coûts irrécupérables

Biais pris en compte

  • Biais de l'apport : Le locataire place son apport (PEA, Assurance-Vie) au lieu de l'immobiliser
  • Biais du cash-flow : Si mensualité > loyer, le locataire épargne la différence chaque mois

Comment lire

  • Ligne noire : Patrimoine du locataire (apport + épargne placés)
  • Lignes colorées : Patrimoine de l'acheteur selon 3 scénarios de valorisation
  • Point d'inflexion : Quand une courbe acheteur croise la ligne locataire = l'achat devient rentable
Clé de lecture : Si toutes les courbes acheteur restent sous la ligne noire pendant 10 ans, vous avez intérêt à rester locataire si vous pensez revendre avant 10 ans.
